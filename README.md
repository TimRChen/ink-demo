# react-cli [![Build Status](https://travis-ci.org/TimRChen/react-cli.svg?branch=master)](https://travis-ci.org/TimRChen/react-cli)


## Install

```
$ npm install -g reactjs-hotload-cli
```

## How to create a react-webpack item
```
$ react-init
```
```
$ npm install
```
```
$ npm run dev
```

## Usage

```js
$ react-init --help

  Usage
    react-init [input]

```
